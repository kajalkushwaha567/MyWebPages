STRING

String -> Sequence of character.

String literal/constant -> Sequence of character enclosed within " "(double quotes).

  • " "  ->  Empty string(string having no character).
  • "3"  ->  String having one character.
  • "kajal"  ->  String having 4 character.

String variable -> In c there is no data type for storing value because string consist of character(s). So in c string is stored in array of character.

String variable declaration

syntax :- datatype varname[size];

Example :- char name[12]; -> Here name is a string variable that can store 11 character long string.

How string is stored in RAM

char name[12]="kajal"; -> Here string variable named name is created and intilized with value "kajal".

NOTE

  • In c string is NULL terminated.
  • In c every string ends with NULL.
  • NULL is a character which ASCII is 0. NULL escape sequence is \0 (\zero).

SOME STRING FUNCTION

strlen -> Finds outs the length of a string.

strlwr -> It converts a string to lowercase.

strupr -> It converts a string to uppercase.

strcat -> It appends one string at the end of another.

strncat -> It appends first n characters of a string at the end of another.

strcpy -> Use it for copying a string into another.

strncpy -> It copy first n characters of two strings.

strcmp -> It compares two strings.

strcmpi -> It compares two strings without regard to case("i" denotes that this function ignores case).

strnicmp -> It compares first n characters of two strings, Its not case sensitive.

strdup -> Used for duplicating a string.

strchr -> Finds out first occurrence of a given character in a string.

strstr -> Finds first occurrenceof a given string in another string.

strnset -> It sets first n character of a string to a given character.

strrev -> It reverses a string.
